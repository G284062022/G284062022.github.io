所属研究室について / About the Lab(拓殖大学工学科高橋研)

本研究室では、コンピュータだけで完結するソフトウェアではなく、 現実世界のハードウェア と組み合わせて動く 「組込みシステム」 の開発を中心に研究を行っています。
センサー、マイコン、通信モジュールなどの電子回路を自作しながら、 その上で動作する制御プログラムやアプリケーションを設計し、 「ハードウェアとソフトウェアを一体として最適化する」ことを目的としています。

組込みシステム、ロボティクス、回路設計、信号処理など、ソフトとハードの両面 からアプローチする点が特徴です。 作って動かすことで理解を深める、実験的で実践的な研究を重視しています。

主な研究テーマ / Research Topics

自動追従型スマートスーツケース

Raspberry Pi × OpenCV × arduino × オムニホイールロボット

  • 顔認識により所有者を検出し、自動で追従するスーツケースロボットを開発。
  • オムニホイールを用いた全方向移動制御を Python で実装。
  • カメラサーボとモータ制御を連携させて、滑らかな追従挙動を実現。

研究スタイル・環境 / How We Work

本研究室では、単に論文を書くことだけでなく、「動くものを作る」 ことを大切にしています。 要件整理から設計、実装、実験、発表まで、一連のプロセスを自分で回すことを目標にしています。

開発環境

  • 言語：Java, Python, TypeScript, JavaScript など
  • フレームワーク：Spring Boot, Vue 3, Streamlit 等
  • プラットフォーム：Raspberry Pi, Linux, macOS
